Ersättning för sjukvård, tandvård eller hjälpmedel vid arbetsskada
Om du har en arbetsskada som gör att du måste betala för tandvård, särskilda hjälpmedel eller vård utomlands, så kan du få ersättning för dina kostnader.
På den här sidan
Kostnader för tandvård
Försäkringskassan kan ersätta kostnader för tandvård, om det här stämmer in på dig:
Din skada eller sjukdom har uppstått på grund av arbetet eller på väg till eller från arbetet.
Behandlingen är nödvändig på grund av arbetsskadan.
Du har ett underlag som visar vad kostnaden är, till exempel ett kostnadsförslag, kvitto eller faktura.

Så mycket kan du få för tandvård
Försäkringskassan ersätter inte alltid exakt det belopp som du betalar hos tandläkaren. Det beror på att vi utgår från så kallade referenspriser för att räkna ut hur mycket du kan få.
Om din tandläkare tar ut ett högre pris än referenspriset, så får du ofta bara ersättning som motsvarar referenspriset.

Kostnader för särskilda hjälpmedel
Försäkringskassan kan ersätta kostnader för särskilda hjälpmedel, om det här stämmer in på dig:
Du behöver hjälpmedlet på grund av en arbetsskada, alltså en skada eller sjukdom som har uppstått på grund av arbetet eller på väg till eller från arbetet.
Du behöver hjälpmedlet för att lindra effekterna av din arbetsskada, och för att klara av din vardag och leva självständigt.
Du har ett underlag som visar vad kostnaden är, till exempel ett kostnadsförslag, kvitto eller faktura.
Du är försäkrad i Sverige.
Du som bor eller jobbar i Sverige är oftast försäkrad i Sverige.
Är jag försäkrad i Sverige?
Du kan ansöka om ersättning för kostnader som du har haft, eller för framtida kostnader. För att kunna få ersättning för framtida kostnader ska det handla om återkommande kostnader som du sannolikt kommer att ha.
Vad är ett särskilt hjälpmedel?
Ett särskilt hjälpmedel är ett hjälpmedel som du har provat ut för just dig, och som du behöver för att kompensera för din arbetsskada. Det kan handla om till exempel specialglasögon, hjälpmedel för att kunna gå eller hörapparater. Det kan också vara utrustning som exempelvis träningsredskap.

Så mycket kan du få för särskilda hjälpmedel
När Försäkringskassan räknar ut hur mycket ersättning du kan få bedömer vi vilken kostnad som har varit nödvändig för dig att betala. Det kan handla om hela kostnaden för ditt särskilda hjälpmedel, eller delar av den.
Om till exempel regionen eller någon annan betalar för hjälpmedlet åt dig, så räknar vi inte det som en nödvändig kostnad för dig.
Förutom kostnader för själva hjälpmedlet kan du också få ersättning för
batterier eller annat som behövs för att det särskilda hjälpmedlet ska fortsätta fungera
kostnader i samband med utprovningen, till exempel resor (det billigaste färdsättet) och besöksavgiften.
Det betyder att du kan ansöka om ersättning för kostnader för särskilda hjälpmedel flera gånger, så länge du fortsätter att ha kostnader för hjälpmedlet.
Kostnader för sjukvård utomlands
Försäkringskassan kan ersätta kostnader för sjukvård utomlands, om det här stämmer in på dig:
Din skada eller sjukdom är en arbetsskada, och har alltså uppstått på grund av arbetet eller på väg till eller från arbetet.
Sjukvården behövs på grund av arbetsskadan, till exempel för att vården inte kan vänta tills du kommer tillbaka till Sverige, eller för att det är för lång väntetid för motsvarande vård i Sverige.
Kostnaden är nödvändig för dig att betala, eftersom ingen annan ersätter den (till exempel din privata försäkring eller någon annan ersättning från Försäkringskassan).

Så mycket kan du få för sjukvård utomlands
Om du får ersättning så betalar Försäkringskassan för mellanskillnaden, alltså det belopp som du betalar extra för vården jämfört med vad du skulle ha betalat i Sverige. 
När du ansöker börjar vi med att titta på om du har rätt till ersättning för vård utomlands enligt de regler som gäller alla, oavsett om du har en arbetsskada eller inte.
Om du har haft kostnader som de reglerna inte ersätter, så går vi vidare och utreder om du kan få ersättning genom arbetsskadeförsäkringen i stället. Det gör att möjligheten att få ersättning beror mycket på din situation, och vilken sorts vård du har fått.
